Phoenix
=======

The code for the Phoenix-R paper.

Dependencies
============

   * Numpy
   * Scipy
   * Cython

How to install
==============

Use anaconda python or install dependencies by hand.

```bash
$ pip install cython
$ pip install numpy
$ pip install scipy
```

Building
========

Simple type

```bash
$ make
```

How to use
==========

See the Example.ipynb and More.ipynb notebook.
